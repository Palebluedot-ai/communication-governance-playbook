# Reusable Kickoff (copy into new channel/thread)

```md
[Kickoff]
Global: Build a portable, local-first memory system that remains consistent across channels/devices.
InScope: <fill current phase>
OutOfScope: <fill non-goals for this phase>
Next-1 (Locked): <single next action>
Unlock condition: say "切阶段到 ..." / "暂停当前，改做 ..." / "重排 roadmap".

Please confirm one:
1) Continue current lock
2) Switch phase (explicit phrase)
```

# Operator Rule
If follow-up questions appear, answer them without changing Next-1 unless unlock condition is explicitly triggered.
